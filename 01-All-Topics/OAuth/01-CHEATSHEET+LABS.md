#### Methodology
```bash
- Search for a specific provider and look for endpoints in the documentation
- response_type=token? -> implicit grant
- No state param? -> CSRF
- OpenID? -> try register + SSRF
- redirect_url? 
    > try arbitrary redirect
    > try path traversal
```
